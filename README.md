# 🧮 Calculator API (TypeScript + Express)

A simple Calculator API built using TypeScript and Express.  
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ✅ Input validation
- ⚡ Fast and lightweight API

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- TypeScript

---

```bash
git clone https://github.com/yashg-tech/Calculator-Api.git
